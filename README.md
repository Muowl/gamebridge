## Pré-requisitos

- **Node.js**

- **NPM**

## Passo a Passo para Rodar o Projeto

- Na  pasta do projeto 'game_bridge', rodar ```npm install```

- Para iniciar o servidor ```node server.js``` (Porta padrão 3000, localhost)
